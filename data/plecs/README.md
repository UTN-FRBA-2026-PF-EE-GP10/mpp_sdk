# PLECS panel lookup tables

**Work in progress**: format and grid density are a first guess; they
will change once the PLECS SEPIC model exists and the plant-vs-plant
comparison is actually run.

Generated by `scripts/export_iv_plecs.py` (regenerate, do not edit).

Each CSV is the I-V curve of the 2x Hissuma PSF10MONO series string
(pvlib De Soto fit, see `mpp_sdk/models/pvlib_adapter.py`) at the
per-panel irradiance in the file name, tabulated on a 1000-point voltage
grid from 0 to V_oc at 25 degC. Wire it into PLECS as a 1-D lookup
table feeding a voltage-controlled current source.

These are the same curves the Python harness simulates against, so a
PLECS run and a `harness/compare_bank.py` run share the same panel by
construction (plant-vs-plant layer of the sim-to-real protocol in
TODO.md).
